# SQL - Operators
* eq
* !eq
* bt
* !bt
* be
* !be
* in
* !in
* like
* !like
* gt
* ge
* lt
* le


## EQUAL
```
{ 'eq' : value }
```

## NOT EQUAL
```
{ '!eq' : value }
```

## BETWEEN
```
{ 'bt' : (low, high) }
```

## NOT BETWEEN
```
{ '!bt' : (low, high) }
```

## BETWEEN - EQUAL THAN
```
{ 'be' : (low, high) }
```

## NOT BETWEEN - EQUAL THAN
```
{ '!be' : (low, high) }
```

## IN
```
{ 'in' : [ list ] }
```

## NOT IN
```
{ '!in' : [ list ] }
```

## MATCH STRING PATTERN
```
{ 'like' : "%pattern%" }
```

## NOT MATCH STRING PATTERN
```
{ '!like' : "%pattern%" }
```

## GREATER THAN
```
{ 'gt' : value }
```

## GREATER OR EQUAL THAN
```
{ 'gt' : value }
```

## LESS THAN
```
{ 'lt' : value }
```

## LESS OR EQUAL THAN
```
{ 'le' : value }
```
